# Three.js 3D 태양계 시뮬레이터

Three.js 기반으로 제작한 웹 3D 태양계 시뮬레이터입니다.

## 구성 파일

- `index.html`
- `src/main.js`
- `src/style.css`

## 실행 방법

아래 방법 중 하나로 정적 서버를 띄운 뒤 브라우저에서 확인하세요.

### 1) VS Code Live Server 사용

1. 프로젝트 폴더를 VS Code로 엽니다.
2. `index.html` 파일에서 **Open with Live Server**를 실행합니다.

### 2) Python 내장 서버 사용

```bash
python3 -m http.server 5173
```

실행 후 브라우저에서 아래 주소로 접속합니다.

- <http://localhost:5173>

## 주요 기능

- 태양 + 8개 행성(수성~해왕성) 표시
- 행성별 자전 및 태양 중심 공전 애니메이션
- 행성 궤도선 렌더링
- OrbitControls 기반 마우스 상호작용(회전/확대/축소)
- 행성 이름 라벨 표시
- 지구 강조(크기/발광)
- 태양 발광 표현 + PointLight
- 별 배경 우주 스카이돔

## 참고

- 실제 천문 비율을 그대로 쓰지 않고 화면 가독성을 위해 축약된 크기/거리 스케일을 사용했습니다.
- 공전/자전 속도는 실제 상대적 차이를 참고해 설정했습니다.
